# Joe T Emulator

A web-based retro game emulator built with Next.js and React. Play classic games directly in your browser with support for multiple gaming systems.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the emulator.

## Features

- Support for multiple gaming systems (NES, SNES, Game Boy, N64, and more)
- Customizable themes
- Cover art support
- Game library management
- Browser-based storage with IndexedDB
